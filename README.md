# 🧮 Calculator Web App

## 👩‍💻 About the Project
This is a simple and interactive Calculator Web Application developed using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations with a clean and modern user interface.

---

## 🚀 Features
- ➕ Addition, Subtraction, Multiplication, Division  
- ⌫ Delete and Clear functionality  
- 🎯 Real-time calculation  
- ⌨️ Keyboard input support  
- 🌙 Dark theme with neon UI  
- ⚡ Smooth button interactions  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## 📂 Project Structure
calculator/
│
├── index.html

---

## 💡 How It Works
- User clicks buttons or uses keyboard input  
- Input is displayed on screen  
- JavaScript processes the expression  
- Result is shown instantly
